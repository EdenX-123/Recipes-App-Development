📘 User Manual – Recipe Web Application
1. Introduction

This Recipe Web Application is designed to help users explore, create, and manage cooking recipes easily. The system provides different categories such as Daily Recipes, Holiday Specials, and Health & Diet options.

Users can browse recipes, create their own recipes, and manage their accounts through a simple and user-friendly interface.

2. Features
🔐 User Authentication

Users can sign up with a username and password

Users can log in and log out securely

🍽 Recipe Management

Add new recipes

Edit existing recipes

Delete recipes

View detailed recipe information

📂 Categories

The system provides the following recipe categories:

🍰 Desserts & Drinks

Includes sweet desserts and beverages

Example: cakes, smoothies, coffee

🎉 Holidays

Special recipes for events such as:

Mother's Day

New Year

🥗 Health & Diet

Focus on healthy eating:

Keto recipes (low-carb)

Vegetarian recipes (plant-based)

3. How to Use the System
📝 Sign Up

Click the Sign Up button

Enter a username (minimum 4 characters)

Enter a password (minimum 6 characters)

Confirm the password

Click Register

🔑 Login

Click the Login button

Enter your username and password

Click Login

➕ Add a Recipe

Navigate to Add Recipe page

Fill in:

Recipe title

Ingredients

Instructions

Image (optional)

Click Submit

✏️ Edit a Recipe

Open a recipe detail page

Click Edit

Update the information

Click Save

🗑 Delete a Recipe

Open a recipe

Click Delete

Confirm the action

🔍 Browse Recipes

Users can browse recipes by category:

Desserts & Drinks

Holidays

Health & Diet

4. User Interface Overview

Navigation bar for easy access

Cards layout for displaying recipes

Clean and responsive design

Popup messages for errors and success notifications

5. Error Handling

The system provides feedback for common errors:

Username already exists

Password too short

Passwords do not match

Invalid login credentials

All error messages are displayed clearly to guide the user.

6. Technical Requirements

Web browser (Chrome, Edge, etc.)

Internet connection (for deployed version)

7. Conclusion

This application provides a simple and effective way for users to manage and explore recipes. It is designed with usability and functionality in mind, making it suitable for everyday cooking needs.
